Python Pamati

Šajā sadaļā jūs atradīsiet informāciju par Python programmēšanas valodas pamatiem.

Sintakse

Python valoda ir izteiksmīga un viegli lasāma. Šeit ir piemērs vienkāršas programmas:


print("Sveika, pasaule!")

Programmas izpildes rezultāts:


Sveika, pasaule!

Datu tipi

Python atbalsta dažādus datu tipus, tostarp:

  • Virknes (strings) - teksta dati, piemēram, "Sveika, pasaule!"
  • Veselie skaitļi (integers) - veseli skaitļi, piemēram, 42
  • Decimālskaitļi (floats) - skaitļi ar decimālpunktu, piemēram, 3.14
  • Saraksti (lists) - saraksti ar vairākiem elementiem, piemēram, [1, 2, 3]
  • Vārdnīcas (dictionaries) - pāru saraksti ar atslēgu un vērtību, piemēram, {'a': 1, 'b': 2}

Uzdevumi un funkcijas

Python ļauj definēt funkcijas, kas veic konkrētas darbības. Šeit ir piemērs funkcijas definīcijai un izsaukumam:


def sveiks(vards):
    print("Sveiki, " + vards + "!")

sveiks("Jānis")

Programmas izpildes rezultāts:


Sveiki, Jānis!
